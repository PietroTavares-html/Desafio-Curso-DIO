Tags

Tags! Primeiramente, o que são tags? As tags HTML são usadas para informar ao navegador a estrutura e o conteúdo do site. Basicamente, as tags são todo o corpo/estrutura de um site, compondo assim aquilo que há de mais importante no site! Nesta fase prendi sobre as funções de cada tag, dentre elas, tags para links, textps, linhas, fontes negrito e italico, entre outras. Trazendo assim conhecimento, e me possibilitando fazer esta atividade paa você!

Listas Ordenadas e Não Ordenadas

Listas Ordenadas: Listas compostas por várias linhas (opcionalmente), sendo elas numeradas conforme a sua necessidade! Exemplo: 1. Item 1 2. Item 2

Listas Não Ordenadas: Listas compoostas por várias linhas (opcionalmente), sendo elas sinalizadas/marcadas por um "•" Exemplo: • Item 1 • Item 2

Textos

Textos basicamente são compostos pela tag "h1" a "h6" tendo assim diferentes tipos de texto. Tendo também "p" sendo usado para definir e separar parágrafos Podemos usar também os comandos "strong" e "i", sendo o "strong" para colocar as palavras em negrito, e o "i" para por as letras em itálico!

Links

a tag links é usada para redirecionamentos, seja para a mesma aba, com outro conteúdo, ou para outra aba" Formamos essa diferença usando, ou deixando de usar o comando "target="_blank" tendo ele a função de redirecionar para outra aba!
